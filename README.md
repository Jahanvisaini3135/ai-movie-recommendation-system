# 🎬 AI Movie Recommendation System

A scalable machine learning–powered movie recommendation system built with **Django** and trained on the **TMDB dataset (1M+ movies)**.  
The system uses **TF-IDF vectorization + Cosine Similarity + SVD optimization** to generate intelligent, content-based movie recommendations.

---

## 📌 Features

- 🎯 Content-based movie recommendation engine
- ⚡ Fast similarity search using Cosine Similarity
- 🧠 TF-IDF + SVD dimensionality reduction
- 🔎 Smart movie search functionality
- 🎛 Advanced filters:
  - Genre-based filtering
  - Rating-based filtering
  - Year range filtering
- 🌐 Django-based full-stack web interface
- 📊 Handles large-scale dataset (1M+ movies)
- 🧾 Clean API-style backend logic
